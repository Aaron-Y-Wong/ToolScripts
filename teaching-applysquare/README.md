# 教学立方课件下载脚本

## Description

在课件页点击导航栏"显示下载链接", 无论课件是否开放下载权限.

由于教学立方网站设计发生改动, 原先的[脚本](https://greasyfork.org/zh-CN/scripts/399058-教学立方课件下载脚本)已经失效, 此脚本在其基础上修改而来, 主要有如下变化:

+ 修正代码逻辑以适用于新的网站结构
+ 异步获取下载链接, 速度更快

## Usage

配合 [Tampermonkey](https://www.tampermonkey.net/) 插件使用.
